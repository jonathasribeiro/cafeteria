# Cafeteria Essência e Sabor

Web application for a coffee shop, built with **React**, **Redux Saga**, **Firebase**, and **styled-components**.

## Features

- Product catalog and order management
- Redux + Redux Saga for async state management
- Firebase integration for backend services
- Toast notifications and responsive layout
- TypeScript for type-safe components

## Stack

| Layer | Technology |
|-------|------------|
| UI | React |
| State | Redux + Redux Saga |
| Backend | Firebase |
| Styling | styled-components |
| Language | TypeScript |

## Getting started

```bash
npm install
npm start
```
